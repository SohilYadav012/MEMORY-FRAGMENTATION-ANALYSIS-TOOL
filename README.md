# Memory Fragmentation Analysis Tool

An interactive web-based simulator that demonstrates how memory is allocated and fragmented in an Operating System.

## 🚀 Features
- Simulates dynamic memory allocation
- Supports allocation algorithms:
  - First Fit
  - Best Fit
  - Worst Fit
- Visual representation of memory blocks
- Real-time calculation of:
  - Internal Fragmentation
  - External Fragmentation
  - Memory Utilization
- Allocate and deallocate processes dynamically
- Toggle between dynamic and fixed partitioning

## 🧠 Purpose
This project helps in understanding how different allocation strategies impact memory efficiency and how fragmentation occurs in real-world systems.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Clone the repository  
2. Open `index.html` in your browser  

## 📌 Use Case
Educational tool for learning Operating System memory management concepts.
